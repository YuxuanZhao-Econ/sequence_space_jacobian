# Sequence-Space Jacobian in Julia

This repository contains Julia notebooks for learning the Sequence-Space Jacobian (SSJ) method from the ground up.

It is a teaching-oriented implementation rather than a production toolkit. The notebooks emphasize transparent model blocks, DAG structure, residual maps, Jacobians, impulse responses, heterogeneous-agent transitions, and the fake news algorithm.

## Requirements

The notebooks are written in Julia and use:

- `LinearAlgebra`
- `Plots`

Run the notebooks from the repository root so that:

```julia
include("SSJ_Function.jl")
```

loads the shared helper functions correctly.

## Introductory Notebooks

- `RBC.ipynb`
  - representative-agent RBC model
  - finite-difference SSJ construction
  - impulse responses to TFP and news shocks

- `KS1998.ipynb`
  - Krusell-Smith style heterogeneous-agent model
  - EGM and distribution iteration
  - block-level SSJ and fake news algorithm
  - terminal distribution diagnostics for finite-horizon transitions

- `SSJ_Function.jl`
  - shared helper functions for the notebooks

## Method

Both notebooks follow the standard sequence-space workflow:

1. Represent the economy as a directed acyclic graph of blocks.
2. Choose unknown sequences and target residuals.
3. Write equilibrium as `H(U, Z) = 0`.
4. Linearize around the deterministic steady state.
5. Solve `G^{U,Z} = -H_U^{-1} H_Z`.
6. Recover other aggregate variables by applying the DAG chain rule.

For heterogeneous-agent blocks, the KS notebook compares direct finite differences with the fake news algorithm for computing household Jacobians.

## Interpretation

The current KS notebook implements first-order SSJ around a deterministic steady state. Aggregate shock paths are treated as perfect-foresight transition paths when solving the household block.

This is useful for local impulse responses and local comparative dynamics. It should not be interpreted as a full stochastic stationary KS economy with aggregate risk in the household state.
